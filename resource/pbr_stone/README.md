此目录存放的是一套供pbr渲染的石头资源。

## preview.jpg
此资源在UE4中的渲染效果预览。

## pbr_stone_mes.bin
模型文件，可使用**geometryv**预览。

## pbr_stone_base_color.dds
基础色纹理。

## pbr_stone_normal.dds
法线纹理。RGB通道存放的是切线空间的XYZ轴法线偏转。

## pbr_stone_aorm.tga
PBR参数纹理。
 - R通道：Ambient Occlusion，环境光遮蔽信息。
 - G通道：Roughness，粗糙度信息。
 - B通道：Metallic，金属度信息。
